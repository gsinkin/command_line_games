command_line_games
==================

Command line implementation of TicTacToe, ConnectFour, and MegaTicTacToe

Setup:

```
virtualenv --no-site-packages games
source ~/virt/games/bin/activate
pip install -r requirements.txt
```

To Run:

Tic Tac Toe:
```
python game.py t3
```

Connect Four:
```
python game.py c4
```

Magic Tic Tac Toe:
```
python game.y mt3
```
